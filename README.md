### Building an Intelligent Recruitment Decision Support System


### Problem Statement

Organizations receive thousands of applications for open positions, making manual screening inefficient, expensive, and prone to bias. Recruiters often spend considerable time reviewing resumes, which can delay hiring decisions and result in qualified candidates being overlooked.

### Solution
This project leverages machine learning and predictive analytics by analyzing applicant characteristics such as **skills,relevant project, years of experience, location match, communication score, and learder score to predict their suitability for a role. Through exploratory data analysis, feature engineering, and classification modeling, the system categorizes applicants into groups such as Best Match for Role, Best Suited for Role, and Will Get Interview.** EDA was conducted to understand applicant trends, identify influential factors, and uncover relationships between candidate attributes and hiring outcomes.The insights i generated from this project can help organizations improve hiring efficiency, reduce manual effort, minimize bias, and identify top talent more effectively. 


### Project Overview
The Applicant Classification System is designed to support data-driven recruitment decisions by reducing manual effort and improving hiring efficiency. Through exploratory data analysis, feature engineering, and classification algorithms, the model identifies patterns associated with successful applicants and enables recruiters to prioritize top talent more effectively.


### Objectives
- Automate applicant screening
- Reduce recruitment time
- Improve hiring efficiency
- Minimize bias in candidate selection
- Support data-driven recruitment decisions

  ### Dataset
- Records: 1,000 applicants
- Features: 14
- Target Variable: label
- Classes:
• Best Match for Role
• Best Suited for Role
• Will Get Interview

 ### Exploratory Data Analysis
- Key insights:
- Applicant experience strongly influences hiring outcomes.
- Candidates with certifications show higher selection rates.
- Certain skills are highly associated with interview invitations.
